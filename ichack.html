IC Hack 2020

February 2020

During this hackathon we implemented a non-conventional video calling platform, using real-time image segmentation to bring multiple parties together in the same screen. This project won us the title sponsor's challenge: The most creative hack that connects people together.

We built the frontend in HTML and vanilla JS, backend in Python and the Flask framework. The WebRTC signalling server is hosted by Scaledrone. The image processing (foreground/background segmentation) is done in python with the OpenCV and TensorFlow Lite libraries.

You can view a demo on YouTube, and our GitHub repository for the project here.
